# Interactive Letter Website

This project is an interactive web experience that simulates opening a letter from a closed envelope. When the user clicks on the envelope, it opens and reveals the letter inside.

## Project Structure

```
interactive-letter-website
├── index.html      # The main HTML file containing the structure of the webpage
├── style.css       # The CSS file defining the styles and animations
├── script.js       # The JavaScript file that adds interactivity
└── README.md       # This documentation file
```

## Getting Started

To set up and run the project locally, follow these steps:

1. **Clone the repository** (if applicable):
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd interactive-letter-website
   ```

3. **Open the `index.html` file** in your web browser:
   - You can simply double-click the `index.html` file or open it using your preferred web browser.

## Features

- A closed envelope that opens when clicked.
- An animated unfolding of the letter inside the envelope.
- Responsive design that works on various screen sizes.

## Technologies Used

- HTML5
- CSS3
- JavaScript
